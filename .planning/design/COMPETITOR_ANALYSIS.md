### Карточка конкурента

| Параметр | Значение |
|:---|:---|
| **Стиль** | Dark mode, космическая тематика (космонавт, звёзды) |
| **Палитра** | Тёмно-синий фон (`~#0a1628`), фиолетовый акцент, pink/gradient CTA |
| **Типографика** | Кастомный bold шрифт для логотипа, стандартный sans-serif для тела |
| **Навигация** | Минимальный sidebar (3 пункта: Заказать, Регистрация, Войти) |
| **UX-паттерн** | Drill-down: Соцсеть → Категория → Услуга → Заказ |
| **Особенности** | Таймер скидки 25%, эмодзи-статусы (⚡🔄🆕), зачёркнутые цены |

### Удачные решения ✅

1. **Таймер «Дадим скидку, чтобы ты остался» (25%)** — отличный urgency trigger, стимулирует конверсию
2. **Drill-down навигация с иконками соцсетей** — интуитивный path: платформа → тип → услуга
3. **Зачёркнутые цены + % скидки** — чёткий визуальный якорь ценности
4. **Минималистичный sidebar** — не перегружен, фокус на действии
5. **Поиск в top bar** — быстрый доступ к услугам

### Ошибки ❌

1. **Космическая тематика — несерьёзно** для B2B, снижает доверие
2. **Нет whitespace в списке услуг** — строки сливаются, трудно сканировать
3. **Фиолетовый на тёмно-синем** — низкий контраст, проблемы с читаемостью
4. **Нет empty states** — при 0 заказах ничего не объясняет
5. **Нет контекста цен** — только «руб/1шт», нет сравнения или spark-trends
6. **3D-маскот (космонавт)** — отвлекает от продукта
7. **Нет визуальной иерархии в списке** — все строки одинаковые

### Карточка конкурента

| Параметр | Значение |
|:---|:---|
| **Стиль** | Light mode, мобильное приложение (mobile-first / PWA) |
| **Палитра** | Белый фон, голубой CTA (`~#00aaff`), голубой glow-border по краям |
| **Типографика** | Крупные заголовки, иерархия выражена хорошо |
| **Навигация** | Hamburger-меню, всего 3 пункта (Услуги, Контакты, FAQ) |
| **UX-паттерн** | Grid иконок соцсетей → Dropdown тип → Цена за 1шт |
| **Особенности** | Визуально выглядит как мобильное приложение, «имитация iPhone» |

### Удачные решения ✅

1. **Grid иконок соцсетей** — визуально мгновенно считывается целевая платформа
2. **Mobile-first подход** — единственный из конкурентов, кто реально mobile-первый
3. **Минималистичный UI** — ничего лишнего, фокус на выборе услуги
4. **Цена за 1 штуку** (а не за 1000) — проще для массового пользователя
5. **Слоган «Ваша аудитория — наша забота»** — эмоциональная связь

### Ошибки ❌

1. **Glow-border вокруг всей страницы** — выглядит как iframe или mockup, не production
2. **Hamburger-меню показывает пустоту** — огромное пустое пространство с 3 пунктами
3. **Нет desktop-оптимизации** — на ПК страница в узкой колонке (mobile-only)
4. **Cookie-popup перекрывает контент** — негативный UX
5. **Нет таблицы цен** — нельзя сравнить услуги между собой
6. **Dropdown вместо таблицы** — скрывает глубину ассортимента
7. **Слишком мало информации на landing** — нет trust signals, отзывов, статистики

### Карточка конкурента

| Параметр | Значение |
|:---|:---|
| **Стиль** | Dark mode, «технологический» (волнистый фон, абстрактные формы) |
| **Палитра** | Тёмно-синий/чёрный фон, белый текст, минимальные акценты |
| **Типографика** | ALL CAPS крупные заголовки, обычный serif/sans-serif для тела |
| **Навигация** | Top bar: Войти, Регистрация, API, Прайс, Обновления, Новости, Поддержка |
| **UX-паттерн** | Tabs (Instagram/VK/Telegram/YouTube/Разное) → Data Table |
| **Особенности** | Полная прозрачность прайса: ID, цена/1000, старт, скорость, мин/макс, списания, детали |

### Удачные решения ✅

1. **Полноценная прайс-таблица open access** — пользователь видит ВСЕ до регистрации
2. **Tab-навигация по платформам** — быстрое переключение
3. **Колонка «Списания»** — честно показывают процент потерь (уникально!)
4. **Колонка «Скорость»** — 50-100/ч, 1000-5000/ч — полная прозрачность
5. **7 столбцов данных** — максимум для экспертного пользователя
6. **API-ссылка в навигации** — явно таргетирует реселлеров

### Ошибки ❌

1. **Дизайн 2018 года** — волнистые SVG-формы, ALL CAPS, нет modern SaaS feel
2. **Data dump** — 7 столбцов таблицы без фильтрации, сортировки, поиска
3. **Нет whitespace** — строки таблицы сжаты, трудно сканировать
4. **Нет виузальных акцентов** — все строки выглядят одинаково
5. **Hero «НАКРУТКА» caps lock** — агрессивный, несовременный тон
6. **Нет CTA в контенте** — кнопка «Запустить» неясна (что за действие?)
7. **Нет мобильной адаптации таблицы** — 7 столбцов → горизонтальный скролл

## 4. 🟢 SMMPrime.com

### Карточка конкурента

| Параметр | Значение |
|:---|:---|
| **Стиль** | Light mode, premium B2B SaaS эстетика |
| **Палитра** | Белый/светло-серый фон, синий primary, оранжевый CTA для контраста |
| **Типографика** | Современный гротеск (Inter/аналог), крупные заголовки, чёткая иерархия |
| **Навигация** | Top bar с dropdown по соцсетям, вход/регистрация |
| **UX-паттерн** | Hero → Калькулятор → Bento-карточки → Таблица услуг |
| **Особенности** | Интерактивный калькулятор прямо на главной, 3D-иллюстрации, нейросетевая тематика |

### Удачные решения ✅

1. **Интерактивный калькулятор на главной** — лучший UX среди всех конкурентов!
2. **Bento-grid карточки** — современный модульный layout
3. **Фильтрация по иконкам соцсетей** в таблице услуг — быстрый доступ
4. **Оранжевый CTA на синем фоне** — контрастный, хорошо заметный
5. **3D-иллюстрации** — профессиональный вид, не клипарт
6. **Строка поиска по ID** — удобно для возвращающихся клиентов
7. **Цена «от 1₽»** — снижает порог входа

### Ошибки ❌

1. **Таблица перегружена** — на мобильных с мелкими колонками неудобно
2. **Одинаковые кнопки «Подробнее»** в каждой строке — визуальный шум
3. **Нет dark mode** — для долгой работы утомительно

### Антипаттерны конкурентов (главные ошибки рынка)

> [!CAUTION]
> **Эти ошибки делают ВСЕ конкуренты. Мы их НЕ повторяем:**

1. **Нет design system** — каждая страница выглядит по-разному
2. **Нет empty states** — пустые экраны без объяснений
3. **Перегруженные таблицы** — 7+ столбцов без возможности скрыть
4. **ALL CAPS заголовки** — агрессивный tone of voice
5. **Космические/абстрактные маскоты** — не вызывают доверие у B2B
6. **Нет micro-animations** — интерфейсы «мёртвые»
7. **Cookie banners перекрывают контент** — UX-катастрофа
8. **Нет responsive tables** — горизонтальный скролл на mobile
