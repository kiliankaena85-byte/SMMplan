## Implementation Decisions

### 1. Visual Integration & Layout (Bento Grid)
- **D-01:** Expand/restructure the Bento Grid in [WhyUs.tsx](file:///d:/SMM_plan_2/src/components/landing/WhyUs.tsx) to feature a dedicated high-contrast, premium, glassmorphic dark-theme card representing the **B2B Reseller Suite**.
- **D-02:** Design the B2B card as a double-span card (`md:col-span-2`) with glowing custom gradient accents (Indigo/Violet/Emerald) to draw immediate attention.
- **D-03:** Re-arrange the Bento Grid layout seamlessly:
  - Row 1: Card 1 (col-span-2: AI-подбор услуг) + Card 2 (col-span-1: Прозрачные условия)
  - Row 2: Card 3 (col-span-1: Персональные скидки) + Card 4 (col-span-2: B2B Reseller Suite & API Hub)

### 2. Marketing Triple-Hook Highlights
- **D-04:** Embed the "Reseller Suite Triple-Hook" showcasing:
  - **Excel Batch Upload:** Native parsing of bulk orders with active link validation.
  - **PerfectPanel API Specs v2:** Plug-and-play synchronization for third-party SMM panels.
  - **Wholesale Price Tiers:** Exclusive wholesale margin rates with up to 15% discount.

### 3. CTA Action & Registration Gateway
- **D-05:** Introduce a styled, premium glowing Button: `"Панель реселлера →"` or `"Начать как B2B Реселлер →"` with smooth transition states.
- **D-06:** Configure the CTA action link pointing directly to `/auth/register?role=reseller&promo=B2BSTART`.
- **D-07:** The downstream registration system will automatically read these URL search parameters to trigger user segment tags and activate early-bird loyalty bonuses.
</decisions>
